# TMSNet

### Environment
Ubuntu 20.04  
SimNIBS 4.5  
PyTorch 2.8.0 (CUDA 12.8)  
MONAI 1.5.1  
python 3.11.10  
wb_command 1.4.2  

### Data
Download from: https://doi.org/10.5281/zenodo.18641851
Extract to the /Data folder.

### Usage
1. Run python main.py
2. Results are saved in /Data/ME01/TMSNet
3. Run optimize.m for optimization results
4. Final output: /Data/ME01/TMSNet/Optimize/CoilCenterCoordinates.csv
